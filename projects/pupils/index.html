pupillometry

an automated pupillometry workflow using machine learning; the manual version of this has already been used in research

During my time at the Harvard Music Lab, I’ve worked on creating a python toolbox to automatically extract and measure pupil size from videos. This tool is currently still being developed, however in the process I’ve built a manual annotation tool which has already been used in a project at the lab that is available as a preprint at the time of writing.

technology used

The toolbox is being developed in python It uses opencv for image manipulation, face_recognition (powered by dlib) to detect facial features in the extracted images and keras, tensorflow and keras-image-segmentation for training a machine learning model on them and interfacing with it.
