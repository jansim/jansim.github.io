coursework

A collection of projects I worked on throughout my undergraduate and graduate studies

Network Analysis (2020)

During the course ‘Network Analysis’ at the University of Amsterdam I created an academic poster analysing the network structure of worry, insomnia and neuroticism.

The full poster was dynamically rendered from a Rmarkdown file and can be downloaded as a PDF here.

Oefenweb (2020)

The below shiny application was created by another student and me in the scope of an assignment for the course ‘Behavioural Data Science Toolbox’ at the University of Amsterdam.

For this project we visualised students’ performance on various learning games / tasks within the ‘oefenweb’ platform. We used the shiny application we’ve built to highlight the fact that 3rd grade students were struggling to properly understand a game on the platform. A particular challenge we had to overcome in this case was to ensure the resulting visualisation always looked good, despite not knowing what data a user may select.

WG-SuchR (2017)

A shiny application for scraping flat prices from a popular website in Germany and showing statistics / the best deals for them. This app was built during a “Learn Shiny” course in my undergrad in collaboration with two other students.

The source code is available on github.
